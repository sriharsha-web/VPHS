# Vignan Public High School (VPHS)

A modern, fast, and visually appealing web application for Vignan Public High School, featuring parallax scrolling, glassmorphism UI components, and premium typography.

## Features

- **Dynamic Scrolling:** Smooth, animated, scroll-based layout using Framer Motion.
- **Glassmorphism UI:** Floating headers and translucent overlapping data cards.
- **Modern Typography:** Utilizing Outfit for headers and Plus Jakarta Sans for high-readability body copy.
- **Responsive Design:** Fully fluid across mobile, tablet, and desktop viewports.

## Running Locally

To run this project locally, ensure you have Node.js and npm installed.

```sh
# Step 1: Clone the repository
git clone https://github.com/sriharsha-web/VPHS.git

# Step 2: Navigate to the project directory
cd VPHS

# Step 3: Install dependencies
npm install

# Step 4: Start the development server
npm run dev
```

## Tech Stack

This project is built with standard, high-performance web tooling:
- Vite
- React
- TypeScript
- Tailwind CSS
- shadcn/ui
- Framer Motion

## Structure

- `src/pages`: Individual route components (Home, About, Admissions, etc.)
- `src/components`: Reusable layout and UI elements, including standard accessible components from shadcn.
- `src/index.css`: Global styling tokens and glassmorphism utilities.
